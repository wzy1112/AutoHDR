- 添加需要监听的游戏exe启动程序，通过监听系统进程列表，当启动对应程序时开启hdr效果，所有程序关闭时关闭hdr效果
- 没要找到win10系统直接开关hdr的api，是通过模拟键盘操作快速打开系统界面点击hdr选项然后关闭窗口来达到效果
- 窗口时间和按钮时间的单位是千分之一秒，控制模拟键盘操作的速度快慢，如果电脑性能第一点，页面加载慢，可以适当调大数值
